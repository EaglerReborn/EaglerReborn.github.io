# List of Globals

  • PluginAPI.player A LocalPlayerData made from EntityPlayerSP.
  • PluginAPI.items An ItemData array of all the in-game item types, and block-item types. [Auto]
  • PluginAPI.blocks A BlockData array of all the in-game blocks. [Auto]
  • PluginAPI.materials A MaterialData array of all the block materials. [Auto]
  • PluginAPI.enchantments An EnchantmentData array of all the in-game enchantments. [Auto]
  • PluginAPI.constructors A object containing constructors to make some Java objects in JavaScript. [Auto]

In order to use non auto globals, they must be required.
